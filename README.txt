usage: Manage qTest Data: [-h] [-v VERBOSE] [-f FILENAME] [-prj PROJECT] [-rl RELEASES] [-cl TEST_CYCLES] [-ts TEST_SUITES]

options:
  -h, --help            show this help message and exit
  -v VERBOSE, --verbose VERBOSE
                        <verbose [error]> Options are debug,info,warning,error Default is error
  -f FILENAME, --filename FILENAME
                        <name of excel file>, .xlsx Output filename (.xlsx)
  -prj PROJECT, --project PROJECT
                        <name of project>, Specify the Project Name
  -rl RELEASES, --releases RELEASES
                        <name of release or rl-xxx>, <[all]> Release hierarchy.
  -cl TEST_CYCLES, --test-cycles TEST_CYCLES
                        <name of test-cycles or cl-xxx>, <[all]> specifing the test cycles hierarchy.
  -ts TEST_SUITES, --test-suites TEST_SUITES
                        <name of test-suites or ts-xxx>, <[all]> specifing the test test suite hierarchy.
